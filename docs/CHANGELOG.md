# Changelog

All notable changes to this project will be documented in this file.

## [0.5.2] - 2025-04-09

### Added
- User authentication system with Flask-Login and JWT
- Guest session support for anonymous users
- Model chain visualization with toggleable details
- Enhanced message formatting in conversation UI

### Changed
- Updated server management scripts to use port 5001
- Improved error handling in server startup
- Updated dependency management to use Python 3.11

### Fixed
- Fixed import issues in authentication modules
- Improved session management reliability
- Fixed message formatting in conversation display

## [0.5.1] - 2025-04-08

### Added
- Initial implementation of user authentication
- Basic server management scripts
- Initial model chain visualization

### Changed
- Updated dependency versions
- Improved error handling

## [0.5.0] - 2025-04-07

### Added
- Initial multi-model support
- Basic conversation management
- Initial UI components

### Changed
- Restructured project layout
- Updated dependency management

## [0.4.0] - 2025-04-06

### Added
